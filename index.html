Installation Guide

Follow these steps to make sure Mobile Security is set up correctly on the device.

⚠️ Important Requirement: Mobile Security works best on a device that is freshly formatted (factory reset) OR has no Google account set up yet. This ensures the app installs and runs properly without conflicts.

1

Prepare the Device

Make sure the client's phone is freshly formatted (factory reset) or has no existing Google / Gmail account linked to it before proceeding.

2

Install Mobile Security App

Download and install the Client APK on the device. Allow all necessary permissions during setup for full functionality.

3

Link to Admin Dashboard

Open the Admin Dashboard (web or APK) and register the device. You can now remotely lock or unlock the client's screen at any time.

4

Configure Parental Controls (Optional)

If using for parental control, set up the child's screen time limits, app restrictions, and content filters from the dashboard.
